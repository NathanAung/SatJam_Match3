## ゲーム説明
シンプルなマッチ3パズルゲームシステムの実装です。<br><br>
6×10のグリッドに4色のボールがランダムに配置され、左クリックでボールを選択し、隣接するボールと入れ替えることができます。ただし、横または縦に3つ以上のマッチが成立する場合のみスワップが有効になります。<br><br>
マッチが成立するとそのボールは消え、上のボールが落下して埋められます。消去後は盤面全体を再チェックし、追加のマッチが発生していれば続けて消去します。<br><br>
ゲーム開始時には、初期盤面にマッチが存在しないようにしつつ、必ずマッチ可能な状態になるよう調整しています。また、消去後にどうしてもマッチが作れない状況になった場合は、盤面をシャッフルしてプレイ可能な状態に戻します。
